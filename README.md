# 📌 Project Overview

This project applies econometric and time-series forecasting techniques to analyze and forecast key macroeconomic and banking indicators in India. The workflow combines statistical modelling, forecasting analytics, stationarity diagnostics, and regression analysis to evaluate macroeconomic trends and banking-related financial indicators.

The project focuses on forecasting inflation trends while analyzing the relationship between inflation, policy rates, exchange rates, money supply, and banking-credit indicators using ARIMA and ARDL models.

---

# 🎯 Project Objectives

* Forecast inflation trends using ARIMA models
* Analyze short-run and long-run macroeconomic relationships using ARDL
* Perform stationarity and residual diagnostics
* Evaluate forecasting performance using statistical accuracy metrics
* Build reproducible forecasting workflows using Python
* Generate banking and macroeconomic insights from time-series data

---

# 📊 Data Sources

The project uses publicly available macroeconomic and banking datasets sourced from:

* Reserve Bank of India (RBI DBIE)
* Ministry of Statistics and Programme Implementation (MoSPI)
* World Bank Open Data
* Federal Reserve Economic Data (FRED)

Due to licensing and repository-size considerations, raw datasets are not uploaded to this repository. The notebooks and scripts are fully reproducible using publicly available data sources.
