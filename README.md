# macroeconomic-forecasting-banking-analytics
Forecasting inflation and banking indicators using ARIMA, ARDL, and econometric modelling with RBI and MoSPI macroeconomic data.


The project uses publicly available macroeconomic and banking datasets sourced from:

- Reserve Bank of India (RBI DBIE)
- Ministry of Statistics and Programme Implementation (MoSPI)
- World Bank Open Data
- Federal Reserve Economic Data (FRED)

Due to licensing and repository-size considerations, raw datasets are not uploaded to this repository. The notebooks and scripts are fully reproducible using publicly available data sources.


| Variable                    | Description                      |
| --------------------------- | -------------------------------- |
| CPI Inflation               | Consumer Price Inflation         |
| Repo Rate                   | RBI Policy Repo Rate             |
| Bank Credit                 | Scheduled Commercial Bank Credit |
| USD-INR Exchange Rate       | Rupee-Dollar Exchange Rate       |
| Industrial Production (IIP) | Industrial Output Indicator      |
| Money Supply (M3)           | Broad Money Supply               |
| Crude Oil Prices            | Global Oil Price Benchmark       |
